Phonics & Numbers Quest
Phonics & Numbers Quest is a console-based classroom game designed to help close basic literacy and numeracy gaps for early primary learners (ages 6-9). Built for real classroom constraints: one shared screen, no individual student devices, and a teacher who plays "game master" where the teacher reads questions aloud, takes the class's answers, and tracks each learner's score on a live leaderboard. The game has two content areas: Word Mode (phonics: sound blending, rhyming, first/last letters) and Number Mode (numeracy: counting, sequencing, basic addition/subtraction), plus a Mixed Mode. Question banks are stored in plain text files, so teachers can edit or expand them without touching the code. This was built using variables, strings, lists, dictionaries, functions, loops, and file reading, and was inspired by my work as a Leadership Associate at Lead For Ghana, where I mentor learners at an underserved primary school in Ghana.
